Flex item ordering using order

order:0 (the default)

<section style="display:flex;">
    <box style="order:0;">Number One</box>
    <box style="order:0;">Number Two</box>
    <box style="order:0;">Number Three</box>
</section>
Number One Number Two Number Three

order:1

<section style="display:flex;">
    <box">Number One</box>
    <box style="order:1;">Number Two</box>
    <box">Number Three</box>
</section>
Number One Number Two Number Three

order:-1

<section style="display:flex;">
    <box">Number One</box>
    <box style="order:-1;">Number Two</box>
    <box">Number Three</box>
</section>
Number One Number Two Number Three
